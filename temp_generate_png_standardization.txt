# Generate PNG Button Standardization
**Date:** October 3, 2025
**Status:**  Completed

## Overview
Converted all "Generate PNG" buttons throughout the application to use camera icon-only buttons, matching the new design pattern established in the Metrics view.

---

## Changes Summary

### Button Transformation
**Before:**
\\\html
<button class="svg-export-btn">
    <span class="codicon codicon-device-camera"></span>
    Generate PNG
</button>
\\\

**After:**
\\\html
<button class="icon-button" title="Generate PNG">
    <i class="codicon codicon-device-camera"></i>
</button>
\\\

### Key Differences:
-  Removed text label "Generate PNG"
-  Added tooltip "Generate PNG" 
-  Changed class from \svg-export-btn\ to \icon-button\
-  Changed \<span>\ to \<i>\ for icon
-  Consistent styling with refresh buttons

## Architecture Notes

This change demonstrates:
1. **Design System Consistency** - Single button style across the app
2. **Iterative Improvement** - Starting with one view, then applying globally
3. **Non-breaking Changes** - Button IDs and functionality unchanged
4. **Accessibility Maintained** - Tooltips provide context for icon-only buttons
